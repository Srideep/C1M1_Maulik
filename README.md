### Project 1 - Statistical Analytics
This project is a simple application that perform statistical analytics on a dataset in C.

The application provides the following statistical analysis functions:
- **Print Statistics**: Displays all calculated statistics (min, max, mean, median)
- **Print Array**: Outputs the complete dataset to the command line
- **Find Median**: Calculates the middle value of the sorted dataset
- **Find Mean**: Computes the average value of all elements
- **Find Maximum**: Identifies the largest value in the dataset
- **Find Minimum**: Identifies the smallest value in the dataset
- **Sort Array**: Orders data from largest to smallest (descending order)
